# 功能与现状

## 打印机管理（核心）
- 打印机列表展示：
  - 打印机名称
  - 端口信息
  - 队列状态（tag）
  - 驱动信息
  - 操作按钮（备份/安装/卸载）
- 状态标签：
  - `就绪`
  - `脱机`
  - `未安装`

## 备份
- 支持按已安装打印机进行驱动备份。
- 备份后写入索引和元数据，供后续安装恢复使用。

## 安装（含向导）
- 自定义安装向导（非 `el-dialog`）：
  - 步骤 1：打印机显示名称校验（非空、重名）
  - 步骤 2（IP 型端口）：IP 输入、格式校验、连通性检测
- 从备份索引恢复驱动并安装。
- USB 场景支持“驱动已安装，等待 USB 重连”的提示。

## 卸载
- 卸载打印机与驱动，回传残留文件信息。
- 卸载期间做操作互斥，避免并发导致状态错乱。

## 实时状态同步
- 主进程启动 `printer-state` 轮询系统状态。
- 状态变更通过 IPC 广播给渲染进程。
- 渲染进程合并到 `PrinterServerManage.printers` 展示。

## 系统设置
- 主题模式：亮/暗/跟随系统。
- 备份目录配置。
- 设置持久化文件：`electron/resource/config/system.json`。

## 当前已知限制
- USB 打印机“安装时立即命名”为系统行为限制场景，自动重命名功能已移除。
- 某些厂商驱动状态字段可能存在差异，需按具体设备补充规则。
