# 项目简介

## 项目名称
- `ele-drive`（桌面端 Electron 应用）

## 项目定位
- 一个面向 Windows 打印机场景的驱动管理工具。
- 主要解决打印机驱动“备份、还原安装、卸载、状态查看”的运维问题。

## 核心价值
- 将驱动备份与恢复流程产品化，减少手工命令操作。
- 将打印机状态（就绪/脱机、端口变化）做运行时同步，提升可视化管理能力。
- 将复杂 PowerShell 脚本集中管理并校验签名，降低维护风险。

## 主要能力
- 打印机列表管理（安装状态、备份状态、可用状态）。
- 打印机驱动备份（生成索引与元数据）。
- 从备份索引安装打印机（含安装向导、IP 检测流程）。
- 打印机/驱动卸载（含残留检查信息）。
- 实时状态同步（主进程 worker 轮询并向渲染进程广播）。
- 系统设置管理（备份目录、主题模式）。

## 技术栈
- Electron（主进程/预加载）
- Vue 3 + Vite（渲染进程）
- Pinia（运行时状态管理）
- Element Plus（UI 组件）
- PowerShell（Windows 打印机相关系统能力）
