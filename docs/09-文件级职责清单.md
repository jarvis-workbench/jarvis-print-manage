# 文件级职责清单（维护范围）

说明：本清单覆盖“项目可维护范围内”的文件与目录，不包含 `node_modules`、构建缓存等第三方生成内容。

## 根目录
- `.git/`：Git 元数据。
- `.idea/`：IDE 配置。
- `.vscode/`：编辑器配置。
- `dist/`：前端构建输出。
- `electron/`：桌面主进程相关代码与资源。
- `public/`：前端静态资源目录。
- `release/`：打包输出目录。
- `src/`：前端业务代码。
- `docs/`：开发记忆库。
- `.gitignore`：Git 忽略配置。
- `index.html`：Vite 前端入口 HTML。
- `package.json`：依赖、脚本、打包配置。
- `package-lock.json`：依赖锁定文件。
- `README.md`：项目对外基础说明（较简版）。
- `vite.config.js`：Vite 配置文件。

## electron
- `electron/main.mjs`：主进程入口，统一管理 IPC 与系统能力。
- `electron/preload.mjs`：桥接 API 暴露给渲染进程。
- `electron/printer-state-worker.mjs`：打印机状态轮询 worker。
- `electron/tray.png`：托盘图标资源。
- `electron/config/`：配置与脚本仓库。
- `electron/resource/`：安装目录 `resources` 下的可写资源（不进 asar）。

## electron/config
- `electron/config/app/.gitkeep`：预留应用配置目录占位。
- `electron/config/persistent/.gitkeep`：预留持久化目录占位。
- `electron/config/script/.gitkeep`：脚本目录占位。
- `electron/config/script/ps/index.mjs`：PowerShell 模板加载与签名校验逻辑。
- `electron/config/script/ps/sign.json`：脚本完整性签名数据。
- `electron/config/script/ps/doc.md`：脚本作用说明文档（当前存在编码问题，待修复）。
- `electron/config/script/ps/*.ps1s`：各打印机功能脚本模板。

## electron/resource
- `electron/resource/config/system.json`：系统配置持久化文件（备份目录、主题模式等）。

## src
- `src/main.js`：前端入口，初始化 Pinia、Router、主题。
- `src/App.vue`：根组件。
- `src/env.d.ts`：前端全局类型与桥接 API 类型定义。
- `src/style.css`：全局样式。
- `src/theme.js`：主题应用逻辑。
- `src/router/index.js`：前端路由。
- `src/stores/runtime.js`：运行时业务状态中心。
- `src/views/*.vue`：页面视图组件。
- `src/components/`：可复用组件目录（当前按需扩展）。
- `src/assets/`：前端资源目录。
