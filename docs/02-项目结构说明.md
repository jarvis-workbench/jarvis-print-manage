# 项目结构说明

## 根目录（关键项）
- `electron/`：Electron 主进程、预加载、后端 worker、脚本与资源。
- `src/`：Vue 渲染进程（页面、路由、状态管理）。
- `public/`：前端静态资源。
- `dist/`：前端构建产物。
- `release/`：安装包输出目录。
- `docs/`：开发记忆库（本目录）。
- `package.json`：脚本、依赖、打包配置。
- `vite.config.js`：Vite 配置。

## electron 目录
- `electron/main.mjs`：主进程入口，负责 IPC、窗口、托盘、设置读写、打印机核心能力调用。
- `electron/preload.mjs`：渲染进程桥接 API（`window.eleDrive`）。
- `electron/worker/printer-state-worker.mjs`：打印机实时状态采集 worker（轮询并上报）。
- `electron/tray.png`：托盘图标。
- `electron/config/`：应用配置与脚本仓库。
  - `electron/config/script/ps/*.scps1`：拆分后的 PowerShell 脚本模板。
  - `electron/config/script/ps/index.mjs`：脚本加载与签名校验。
  - `electron/config/script/ps/sign.json`：脚本防篡改签名清单。
  - `electron/config/app/`：预留应用配置目录。
  - `electron/config/persistent/`：预留持久化目录。
- `electron/resource/`：打包后复制到安装目录 `resources`（非 `app.asar`）的可变资源目录。
  - `electron/resource/config/system.json`：系统配置持久化文件（当前存放备份目录、主题）。

## src 目录
- `src/main.js`：渲染进程入口，初始化 Pinia/Router/主题。
- `src/App.vue`：应用根组件。
- `src/router/index.js`：页面路由。
- `src/stores/runtime.js`：运行时状态中心（当前核心对象：`PrinterServerManage` + `settings`）。
- `src/views/`
  - `HomeView.vue`：主页。
  - `PrintersView.vue`：打印机管理页（核心业务页面）。
  - `SettingsView.vue`：系统设置页。
  - `DriverInstallView.vue`：驱动安装相关视图。
- `src/theme.js`：主题模式应用逻辑。
- `src/env.d.ts`：桥接 API 与数据结构类型声明。

## 打包资源落位（重要）
- `electron/resource/**` 不进入 `app.asar`。
- 打包后目标为：`安装目录/resources/**`。
- 该规则在 `package.json -> build.files + build.extraResources` 中定义。
