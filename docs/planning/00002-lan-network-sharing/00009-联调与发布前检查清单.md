## 1. 联调环境 Gate
- [ ] 至少 3 台节点（同网段 IPv4）可稳定运行 24h。
- [ ] 每台节点都具备可安装打印机样本。
- [ ] 防火墙端口策略已固定（服务口 + 发现口）。

## 2. 配对鉴权 Gate
- [ ] 未配对节点无法拉取 offers。
- [ ] 配对码 120 秒失效策略生效。
- [ ] 拉黑节点后旧 token 立即失效。

## 3. 传输安装 Gate
- [ ] 可重试错误可在 3 次内恢复。
- [ ] 不可重试错误首次失败即终止。
- [ ] 降级路径可用（手工向导安装）。

## 4. 日志与可观测性 Gate
- [ ] `lan-task-log.jsonl` 可按 `taskId` 回放完整链路。
- [ ] `RETRY_*` 事件完整，次数与退避时间正确。
- [ ] 日志轮转与 14 天清理策略通过验证。

## 5. 回滚 Gate
- [ ] 关闭局域网开关可停止监听与广播。
- [ ] 局域网失败不影响本地备份/本地安装。
- [ ] 可一键清空白名单与局域网日志。
