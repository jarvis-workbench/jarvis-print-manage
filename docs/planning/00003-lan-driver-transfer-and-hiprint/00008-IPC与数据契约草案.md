更新时间：2026-03-28
状态：草案（待评审）

## 1. 新增 IPC 通道
- `print:service:get-state`
- `print:service:set-enabled`
- `print:service:get-client-info`
- `print:service:get-printer-list`
- `print:service:submit-job`
- `print:service:get-job`
- `print:service:list-jobs`
- `print:service:reprint`

## 2. 数据结构

```json
{
  "enabled": true,
  "port": 17521,
  "socketProtocolVersion": 1,
  "running": true,
  "clients": 0,
  "updatedAt": "2026-03-28T12:00:00.000Z"
}
```

```json
{
  "taskId": "print-task-uuid",
  "templateId": "biz-001",
  "type": "html|pdf|url_pdf|blob_pdf|render-jpeg|render-pdf|render-print",
  "status": "QUEUED|RUNNING|DONE|FAILED|CANCELED",
  "printer": "Printer Name",
  "errorCode": "",
  "errorMessage": "",
  "createdAt": "2026-03-28T12:00:00.000Z",
  "updatedAt": "2026-03-28T12:00:01.000Z"

```

## 3. `print:service:submit-job`
- 入参：
```json
{
  "templateId": "biz-001",
  "type": "html",
  "printer": "Printer Name",
  "html": "<html>...</html>",
  "options": {}
}
```
- 返回：`{ taskId: string, status: 'QUEUED' }`

## 4. 广播事件
- `print:job-updated`
- `print:service:state-updated`

## 5. 错误码基线
- `SOCKET_AUTH_FAILED`
- `SOCKET_PROTOCOL_MISMATCH`
- `PRINT_PRINTER_NOT_FOUND`
- `PRINT_PRINTER_UNAVAILABLE`
- `PRINT_EXEC_FAILED`
- `RENDER_TEMPLATE_INVALID`
- `RENDER_SNAPSHOT_FAILED`
- `RENDER_PDF_FAILED`
- `PAYLOAD_INVALID`
- `PAYLOAD_TOO_LARGE`

## 6. 契约约束
- 返回值必须结构化，禁止仅返回字符串错误。
- `taskId` 必须可用于日志回放。
- IPC 与 Socket 事件使用同一错误码集合。
