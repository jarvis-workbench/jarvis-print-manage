# 实施前置清单：局域网共享（P2/P3）

更新时间：2026-03-28
状态：评审中（等待 P1 完成）
关联规划：`docs/planning/局域网组网与驱动共享规划.md`

## 1. 首版边界冻结
- 仅支持同网段 IPv4。
- 仅支持局域网内明文 HTTP + 一次性配对码 + 白名单。
- 不做跨子网、中继、TLS、断点续传。

## 2. 依赖门槛（必须满足才进入开发）
- `P1` 已完成并提供稳定 `archive*` 索引字段。
- 单机安装链路可接受“外部传入压缩包解压目录”作为输入。
- 设置页已有“局域网组网开关”容器位（可先隐藏开关，不暴露给最终用户）。

## 3. 阶段 Gate（防止提前扩 scope）

### Gate A：发现层
- 能发现同网段节点并维护在线状态。
- 至少三台机器 24h 无明显离线抖动。

### Gate B：只读展示
- 能展示“他机有且本机无”的打印机列表。
- 去重优先使用稳定身份字段，名称仅兜底。

### Gate C：传输安装
- 单任务手动触发传输安装闭环跑通。
- 出错链路具备节点/传输/安装三段日志。

## 4. 协议草案（供后续编码对齐）
- `GET /lan/nodes`：节点列表
- `GET /lan/offers`：远端打印机能力
- `POST /lan/install-request`：请求对端准备并发送驱动包
- `GET /lan/tasks/:taskId`：任务状态与错误信息

## 5. 关键风险补丁
- 防误装：安装前强校验 `environment/arch`。
- 防污染：局域网失败不影响本地备份与本地安装。
- 防失控：所有网络能力跟随“局域网组网开关”一键启停。

## 6. 下轮产出预期
- 当 P1 标记 `已完成` 后，产出局域网模块拆分：
  - 进程职责图（main/renderer/worker）
  - 端口与生命周期设计
  - 任务状态机图（discover -> offer -> transfer -> install）
