## 1. 目标
- 在正式发布前演练“功能开关回滚 + 数据清理回滚 + 任务终止回滚”。
- 确认回滚后本地打印相关能力不受影响。

## 2. 演练前提
- 版本：`...`
- 环境：`...`
- 参与节点：`...`
- 演练时间：`...`
- 负责人：`...`

## 3. 演练步骤
1. 启用局域网开关，发起至少 1 个安装任务。
2. 任务执行中触发回滚：关闭局域网开关。
3. 验证服务监听与发现广播是否停止。
4. 执行“清空局域网记录”（白名单 + 日志）。
5. 验证本地备份、安装、卸载链路仍可用。

## 4. 结果记录模板
- Step-1：`通过/失败`，证据：`...`
- Step-2：`通过/失败`，证据：`...`
- Step-3：`通过/失败`，证据：`...`
- Step-4：`通过/失败`，证据：`...`
- Step-5：`通过/失败`，证据：`...`

## 5. 回滚判定标准
- 开关关闭后 5 秒内停止网络能力。
- 进行中任务可进入 `CANCELED` 或 `FAILED`，不可卡住。
- 清空记录后再次启动可重新配对。
- 本地（非局域网）打印功能不回归。

## 6. 复盘模板
- 结论：`通过 / 有条件通过 / 不通过`
- 问题清单：
  - `编号 + 现象 + 根因 + 修复计划`
- 风险等级：`高 / 中 / 低`
- 是否允许发布：`是 / 否`
